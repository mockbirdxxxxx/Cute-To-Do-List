# ✨ Cute To-Do List App ✨

A beautiful, colorful to-do list application with a playful cute theme. Organize your tasks with style and enjoy a delightful user experience!


## How to Use 🚀

1. **Open the App**: Open `index.html` in your web browser
2. **Add a Task**: 
   - Type your task in the input field
   - Press Enter or click the "Add +" button
3. **Complete a Task**: Click the checkbox next to a task to mark it as complete
4. **Delete a Task**: Click the × button on the right side of a task
5. **Filter Tasks**: Use the filter buttons to view All, Active, or Completed tasks
6. **Clear Completed**: Click "Clear Completed ✨" to remove all finished tasks
